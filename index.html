Ubuntu has many community editions, with currently 14 spins. 7 official, 7 unoffical. This page is a list of all of them with relevent links! This is in no particular order (except maybe being lazy when adding new ones)

The difference between remixes and flavors is that
Remixes are NOT offically recognized by Canonical and
Flavors (or Flavours) are recognized by Canonical as quality redistributed versions of Ubuntu
